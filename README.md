# 💱 Currency Converter Web App

A simple and responsive Currency Converter web application built using **HTML, CSS, and JavaScript**. The application fetches real-time exchange rates using the CurrencyAPI and converts the entered amount into multiple international currencies.

---

## 🚀 Features

* 🌍 Convert currencies using live exchange rates.
* 💰 Supports multiple currencies.
* ⚡ Fetches real-time exchange rates using CurrencyAPI.
* 📱 Responsive user interface.
* 🎨 Simple and clean design.
* 🔄 Dynamically updates conversion results without reloading the page.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* CurrencyAPI

## ⚙️ How It Works

1. Enter the amount to convert.
2. Select the base currency.
3. Click the **Submit** button.
4. The application fetches the latest exchange rates from CurrencyAPI.
5. Converted values are displayed in a table.

---


## 🔑 API Used

This project uses the CurrencyAPI service to retrieve real-time exchange rates.

> **Important:** Never expose your API key in a public GitHub repository. Store it securely when deploying production applications.
